# AI Disclosure

## Statement

This repository includes substantial code and documentation produced with AI assistance.

## What AI contributed

- Language and compiler bootstrap scaffolding (`vibc` frontend, parser, IR dump path).
- SDL runtime integration scaffolding and native runner glue for example games.
- Project documentation, usage guides, and build scripts.

## Human responsibilities

- Defining goals and product direction.
- Running/testing builds and giving behavioral feedback.
- Deciding what changes are accepted and shipped.

## Why this is documented

We explicitly disclose AI involvement so users and contributors can evaluate:

- code provenance,
- trust assumptions,
- and review expectations.

## Contributor expectation

All contributions (AI-assisted or human-only) should be reviewed for:

- correctness,
- maintainability,
- security implications,
- and licensing compatibility.
